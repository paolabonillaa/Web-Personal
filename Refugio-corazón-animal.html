Web para Refugio Animal

Cliente

Proyecto no real

Fecha

Febrero 2023

Cargo

Diseñador UX general

El problema:

Corazón animal es un refugio para gatos y perros en condición de calle, que necesita un portal web para que las personas interesadas puedan adoptar, colaborar, donar y realizar citas en veterinaria sin la necesidad de ir al refugio.

La meta:

Diseñar una página web sencilla que le permita a los usuarios encontrar información sobre el refugio, así como poder realizar procesos de adopción, donaciones y solicitar citas veterinarias.

En la página de inicio encontraremos sugerencias de mascotas en adopción, así como información y fotografías sobre el refugio y una sección para recibir información sobre mascotas en situación de calle o maltrato.

En todas las páginas de la web se puede visualizar de dos maneras el perfil de usuario mediante la barra blanca lateral o con la opción "Perfil" en el header.

En el lado izquierdo del header encontramos el logo del refugio el cual cumple la función de botón de inicio, seguido de este tenemos opciones para facilitar el recorrido del usuario como "Adoptar", "Donar", "Veterinario" y "Perfil"; al otro extremo del header tenemos un buscador y las opciones de iniciar sesión o una crear cuenta.

Para lograr adaptar el diseño de desktop a móvil en la página de adopción se empezó reorganizando los filtros de búsqueda, en lugar de estar alineados de forma horizontal se decidió, que para los usuarios sería más cómodo utilizarlos si se encontraban alineados vertical.

Posteriormente se evaluaron diferentes maneras para organizar las mascotas en adopción, pero para mejorar la visualización de cada tarjeta se prefirió alinearlos de forma vertical uno debajo del otro, de esta manera simplemente con realizar scroll se detallará la disponibilidad.

Adopta desde la opción en el hearder para visualizar todas las opciones disponibles para ti o como el ejemplo en video mediante el botón "adopta a", solo debe verificar la información de la mascota y si es la adecuada para ti selecciona el botón "Adoptar" a continuación encontraras un resumen de tu selección más un cuestionario obligatorio con preguntas que le interesan al refugio para la protección de la mascota.

Mediante la web la personas interesadas por la causa podrán realizar sus donaciones, desde la comodidad de sus hogares.

Para brindarles comodidad a los usuarios se implementó un sistema de citas, en el cual los interesados deben colocar la fecha, mes y hora de su interés para posteriormente desde el refugio confirmar disponibilidad mediante correo electrónico.

Todos los visitantes a la web van a tener opción a iniciar sesión o crear una cuenta, con datos básicos como teléfono y correo electrónico.
